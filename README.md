## Requirements ##

- AWS CLI
- Node.js & NPM
- The Bash Shell
  - In Windows 10, see the [Windows Subsystem for Linux](https://docs.microsoft.com/en-us/windows/wsl/install) to get a Windows-integrated version of Ubuntu and Bash.

## Setup & Usage ##

1. `npm run configure`
2. `npm run s3-bucket`
3. `npm run build-layer`

### Deployment ###

1. `npm run deploy`

### Testing ###

1. `npm run invoke`

### Clean-Up ###

1. `npm run clean`
